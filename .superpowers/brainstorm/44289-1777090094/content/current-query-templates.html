Current Ask the Stats query templates

4 question shapes the parser currently understands. Which of these would you like to expand, or should we add entirely new patterns?

1. Count Threshold
How many times has [player] recorded [threshold]+ [stat]?

Counts how often a single player reaches a stat threshold. Does NOT support multi-player or team comparisons.

2. Single Peak
What is [player]'s highest [stat] total against [opponent]?

Finds the maximum stat value for one player, optionally filtered by opponent.

3. Player List
Which players had [threshold]+ [stat] in [season]?

Returns all players in a season meeting a stat threshold, ranked by that stat.

4. Team Low Mark
Which teams had the lowest [stat] in [season]?

Returns teams ranked by a stat in ascending order (lowest first) within a season.
