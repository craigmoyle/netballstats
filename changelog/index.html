25 Jun 2026

Archive UX polish across domestic and international surfaces.

A broad editorial pass on navigation, filters, loading states, citations, and share previews so the archive reads as one coherent reference desk rather than a set of stitched tools.

  • Domestic and international archive homes now use sticky filter rails, debounced auto-apply, and clearer season scope labels. Shared links that mean the full archive now carry ?seasons=all so recipients see the same slice.

  • Compare, round recap, query, nWAR, league composition, and player dossiers no longer paint loading copy into editorial shells before data arrives. Compare adds a recoverable error banner with Try again when builder or comparison data fails.

  • Major pages now ship page-specific Open Graph and Twitter card images, built from scope-branded share artwork at deploy time.
  • New international API endpoints support season-total and trend series for competition, team, and player panels on the international home.

26 Apr 2026

Ask the Stats Query Expansion

Significant expansion of Ask the Stats query capabilities with intelligent parsing and guided builder for complex multi-filter queries.

  • New Query Shapes: Comparison (2-way team/player matchups), Combination (multi-filter AND/OR), Trend (multi-season tracking), Record (all-time rankings)
  • Intelligent Parser: Confidence-based natural language processing with helpful error messages and rephrasing suggestions for ambiguous queries
  • Query Builder: Multi-step guided interface for complex queries with pre-filled hints and suggestions from parser analysis
  • Query Parity: Builder-composed queries produce identical results to natural language equivalents, ensuring consistency
  • Expanded Examples: 8–10 template queries covering all available shapes and use cases for quick exploration
